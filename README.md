# Byjus-Learning-Analytics-Dashboard
Interactive Learning Analytics Dashboard built using Microsoft Excel.

## Project Overview
This project is an interactive Microsoft Excel dashboard designed to analyze student learning performance, course completion, quiz scores, and revenue. It helps visualize key learning metrics for better decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting

## Dashboard Features
- Total Students
- Total Revenue
- Average Quiz Score
- Course Completion Rate
- Performance Rating Analysis
- Gender-wise Analysis
- Interactive Slicers

## Key Insights
- Analyzed student performance and learning progress.
- Compared quiz scores and course completion rates.
- Tracked revenue and payment status.
- Created an interactive dashboard for easy analysis.
