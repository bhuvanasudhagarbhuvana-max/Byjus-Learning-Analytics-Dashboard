# Byju's Learning Analytics Dashboard

## Project Overview
This project is an interactive Microsoft Excel dashboard built to analyze student learning performance, course completion, quiz scores, and revenue. The dataset was cleaned and transformed using Power Query before creating the dashboard.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Data Cleaning (Power Query)
- Imported the raw dataset into Power Query.
- Removed blank rows and duplicate records.
- Corrected data types (Date, Number, Text).
- Handled missing values.
- Renamed columns for better readability.
- Loaded the cleaned data into Excel for analysis.

## Dashboard Features
- Total Students
- Total Revenue
- Average Quiz Score
- Gender-wise Analysis
- Performance Rating Distribution
- Interactive Slicers

## Skills Demonstrated
- Data Cleaning using Power Query
- Data Transformation
- Pivot Table Analysis
- Data Visualization
- Dashboard Design
- Business Reporting
